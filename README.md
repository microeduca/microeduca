# MicroEduca

Aplicação React + Vite para gestão e reprodução de conteúdos de treinamento com integração Vimeo e backend Node/Express + PostgreSQL (Railway).

## Desenvolvimento

Requisitos: Node.js e npm

```sh
npm i
npm run dev
# Em outro terminal
npm run api
```

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Express + pg
